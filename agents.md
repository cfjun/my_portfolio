# 个人作品集网站开发指令

## 项目概述
- 项目名称：个人作品集网站
- 项目描述：展示个人项目、技能、联系信息等
- 项目技术栈：React + TypeScript + Vite
- 项目数据库：JSON 文件（项目数据、技能数据）

## 开发规范
- 使用函数式组件（React Hooks）
- 组件化开发，每个组件负责一个功能模块
- 采用Tailwind CSS进行样式设计
- 遵循React的生命周期方法，如 `useEffect`、`useState` 等
- 数据从JSON文件中读取，避免直接在组件中硬编码
- 组件化要可复用，避免重复代码
- 代码要有注释，方便后续维护和扩展

## 设计要求
- 深色主题：网站主要采用深色主题，避免使用浅色主题
- 使用渐变色作为强调色
- 添加平滑的滚动动画
- 确保移动端和PC端的正常显示和操作

## 注意事项
- 保持设计简洁
- 性能优化：图片使用懒加载
- 确保所有链接可点击
